# MCP Tool Usage Guidelines

These tools are available but should only be invoked when **explicitly needed** for the task at hand. Do not call them proactively, speculatively, or as a default first step.

- **context7** (`context7_*`): Use only when you need to look up documentation for a specific library or framework. Resolve the library ID first, then query docs.
- **gh_grep** (`gh_grep_*`): Use only when you need real-world code examples from GitHub repositories to understand an unfamiliar API or pattern.
- **drawio** (`drawio_*`): Use only for diagram-related tasks — creating or editing Draw.io diagrams. Do not invoke for general coding tasks.
- **playwright** (`playwright_*`): Use only for browser automation, web scraping, or UI testing tasks. Do not invoke just to fetch a URL if a simpler tool suffices.
