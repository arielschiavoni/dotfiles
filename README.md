# dotfiles

##  Install

### Darwin (MacOS)

```bash
cd ./install/darwin/
./install.sh
```

### Linux

```bash
cd ./install/linux/
./install.sh
```

##  Configure

The config script automatically detects the OS and creates the corresponding configuraiton for
the tools relevant to the OS. It uses `stow`.

```bash
cd ./config/
./config.sh
```
