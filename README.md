# dotfiles


## Create symlinks with stow

```sh
$ stow --target ~/Library/ApplicationSupport/Spectacle/ spectacle
$ stow --target ~/Library/ApplicationSupport/Code/User/ vscode
$ stow --target ~ git tig fish
```
